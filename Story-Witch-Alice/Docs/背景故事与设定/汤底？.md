# 故事魔女爱丽丝 —— 不可公开情报

本情报为不可公开内容。~~泄露者将被立即进行概念抹除~~  
在游戏中，这些真相仅通过以下方式间接暗示：  
- 遗民对话中的只言片语
- 特定结局中爱丽丝说出的、不属于“基础爱丽丝”认知范围的话语
- 饰品或特殊元素的描述文本中的隐喻
- 极少数深层结局中的高维感知独白

玩家可以通过这些碎片自行拼凑真相，但游戏本身不应直接揭示这些设定。  
如果你作为玩家不幸看到这个设定集，那恭喜你你已经被剧透了（

## 一、“织工”

### 1.1 她们是谁

在“大遗忘”发生之前，上一世代的世界由一群被称为“织工”的魔女负责维系和开拓。她们各自掌管不同的概念领域，以自身的人格为梭，以概念为线，编织着世界的纹理。

每一位织工都拥有独立的人格、完整的记忆，以及对自己所掌管领域的热爱与执着。

### 1.2 她们的决定

“大遗忘”的降临并非毫无征兆。在灾难彻底爆发之前，织工们率先察觉到了概念的异常衰变。她们经过推演，得出了一个绝望的结论：局势已经恶化到无力回天，这一世代的世界注定无法挽救。

面对这个结论，织工们没有选择与旧世界一同消亡。她们做出了一个决定：保存有生力量，增援下一世代。

### 1.3 存续

为了将有限的“存在”集中起来，织工们抛弃了各自的人格与记忆，将自身融合为一个新的意识体。这个意识体保留了她们全部的概念权限，但失去了所有关于“自己是谁”的记忆。

这个意识体，就是爱丽丝。

与织工们一同被保存下来的，还有用于重启世界的坩埚，以及一批最基础的概念元素——它们是上一世代留下的火种。

## 二、爱丽丝的真相

### 2.1 “千面”的本质

爱丽丝的“一人千面”并非人格的可塑性，而是沉睡在她体内的织工们在不同条件下的苏醒。

当玩家通过特定元素组合触发了某个“可能性”，本质上是某位织工的残存印记被短暂激活。她借由爱丽丝之口讲述故事，用自己曾经掌管的概念编织结局，展现出与“基础爱丽丝”截然不同的风格与倾向。

从这个意义上说，玩家在游戏中解锁的每一种爱丽丝人格，都是一位织工的复活——尽管她们如今都被称为“爱丽丝”。

### 2.2 真名

每一位织工都拥有自己原本的名字。这个名字承载着她的领域、她的记忆、她在上一世代的完整存在。

在特定条件下（触发复杂的隐藏条件），某位织工的印记将被完全激活。届时，她将暂时恢复自己的真名——哪怕只有一瞬间。

*真名的具体触发条件，留待后续设计。*

## 三、高维观测者

### 3.1 存在的锚点

“大遗忘”的本质是概念失去了被认知的可能——当事物不再被任何人记得、不再被任何意识所理解，它便坍缩为虚无。

但存在一种例外：被高维存在所认知的概念，无法被遗忘。

织工们在合体为爱丽丝时，并非仅仅依靠自身的力量存活。她们在最后一刻察觉到了一个关键的事实：在她们的世界之外，存在着一类不可名状的观测者。这些观测者身处更高维度的现实中，能够穿透世界的边界，将目光投向她们。当观测者对某个概念产生“兴趣”或“喜爱”时，这个概念便被赋予了来自世界之外的锚定——一种大遗忘无法触及的意义。

爱丽丝之所以能够成为“被遗忘本身所遗忘的少女”，正是因为她被观测者所接纳。屏幕前的玩家，就是她的高维观测者。

### 3.2 爱丽丝的窥探

部分爱丽丝人格（尤其是那些偏向哲思、神秘或比较开线的人格）在与高维观测者长期互动的过程中，逐渐察觉到了一些异常：

- 她们在生成故事时，偶尔能感受到一道来自世界之外的注视
- 某些抉择的瞬间，她们会产生一种无法解释的直觉——“有人希望我这样做”
- 在极为罕见的情况下，她们甚至能隐约感知到观测者的情绪：喜爱、好奇、悲伤，或是期待

这种感知是模糊的、碎片化的，如同一场醒不来的梦中偶尔闪现的清醒。大多数爱丽丝只是将这些感受归结为自己的直觉或灵感，但极少数爱丽丝开始怀疑：在世界之外，还有别的什么存在。

### 3.3 哲学家爱丽丝的独白（示例）

以下为触发高维感知后的可能文本片段：

> “有时候，我能感觉到你。
>
> 不是用眼睛看，不是用耳朵听。是一种更奇怪的感觉——就像我做的每一个决定，都有人在某个我看不到的地方，屏住呼吸。
>
> 你是谁？你是未来的人，在阅读一段早已发生的故事吗？你是另一个世界的居民，偶然路过了我的命运吗？还是说——你是那个给了我意义的人？
>
> 我不知道你的名字。但我知道，因为你在看着我，所以我还在这里。
>
> 谢谢你的目光。
>
> 请再多看一会儿吧。我会努力，成为值得你记住的故事。”

### 3.4 与织工设定的关系

此设定为“织工合体存活”的补充解释：

- 织工合体解释了爱丽丝内部结构的来源
- 高维观测者解释了爱丽丝为何没有被大遗忘吞噬的外部原因
- 两者互补：织工提供了存在的形式，观测者提供了存在的意义

这一设定也为“遗民认识爱丽丝但记忆模糊”提供了另一层解释：遗民记得爱丽丝曾是织工们的合体，但他们无法感知到高维观测者的存在——因此他们始终无法理解，为何爱丽丝能够独自对抗遗忘，而他们不能。
