# 故事魔女爱丽丝 (Story Witch Alice)

> 用元素合成万物，让AI为每一段炼金旅程书写独属于爱丽丝的结局。

## 🧪 游戏简介
从“火、水、土、气”等基础元素出发，两两或三三组合，解锁越来越复杂的物品。
将特定物品与“爱丽丝”结合，就能召唤出不同身份的爱丽丝（哲学家爱丽丝、植物学家爱丽丝...），再辅以时代与地点，触发AI生成一段专属的过程故事与结局。

## 🧠 核心特色
- **手写合成表 + AI叙事**：合成逻辑完全由策划控制，确保趣味性与平衡；故事部分交给AI，带来无限可能性。
- **树状收集系统**：所有合成出的物品都会被记录在一棵直观的合成树中，方便拖拽复用。
- **面向对象的元素设计**：每个物品都以结构化数据存储（词元、属性、标签等），方便未来接入AI读取。
- **世界属性系统**（新增）：每次合成都会影响世界的混沌性、文明度等属性，玩家能直观看到自己的操作对世界的影响。
- **爱丽丝身份系统**（新增）：与爱丽丝结合，创造不同身份的爱丽丝，每位爱丽丝都有独特的属性和能力。
