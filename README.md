# 故事魔女爱丽丝 (Story Witch Alice)

> 用元素合成万物，让AI为每一段炼金旅程书写独属于爱丽丝的结局。

## 🧪 游戏简介

从“火、水、石、气、时间”等基础元素出发，两两或三三组合，解锁越来越复杂的物品。将特定物品与“爱丽丝”结合，就能召唤出不同身份的爱丽丝，再辅以时代与地点，触发AI生成一段专属的过程故事与结局。

## 🧠 核心特色

- **手写合成表 + AI叙事**：合成逻辑完全由策划控制，确保趣味性与平衡；故事部分交给AI，带来无限可能性。
- **书宠交互系统**：合成表以魔法书的形式呈现，鼠标悬停时书页翻动，点击展开完整的物品栏与合成树。
- **面向对象的元素设计**：每个物品都以结构化数据存储（词元、属性、标签等），通过CSV批量导入，方便维护与未来接入AI。
- **物理拖拽合成**：元素以2D物理物体形式拖拽入锅，支持碰撞、抛物线惯性、重复拾取，手感自然。
- **饰品系统（预留）**：特定结局解锁饰品，穿戴在爱丽丝立绘上，作为故事的纪念品。

## 🛠 技术栈

- 引擎：Unity (C#)
- 合作：GitHub
- 数据管线：CSV批量导入ScriptableObject（ElementData + RecipeData）
- 未来接入：AI文本生成API
